FTD Dashboard Deployment Information
====================================

GitHub Repository: https://github.com/adm7777777/ftd-dashboard
Created: September 9, 2025

Streamlit Deployment:
--------------------
1. Go to: https://share.streamlit.io
2. Sign in with GitHub
3. Click "New app"
4. Enter:
   - Repository: adm7777777/ftd-dashboard
   - Branch: main
   - Main file path: ftd_dashboard.py
5. Click "Deploy"

Your app URL will be something like:
https://ftd-dashboard.streamlit.app
or
https://adm7777777-ftd-dashboard.streamlit.app

The app will take 2-3 minutes to deploy initially.

To update the app:
-----------------
Just push changes to GitHub:
cd ~/Desktop/ftd-dashboard
git add .
git commit -m "Update message"
git push

The Streamlit app will automatically redeploy with your changes.

Local Testing:
-------------
cd ~/Desktop/ftd-dashboard
python3 -m streamlit run ftd_dashboard.py
